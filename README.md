1. OVERVIEW

This privacy policy describes how the Timnot application
("App") handles user data. By installing or using the App,
you agree to this policy.

2. DATA COLLECTED

Timnot does not collect any personal data, usage statistics,
location information, or identifiers of any kind.

The App only stores the following data locally on your device:

  • Tasks (name, action, schedule):
    %APPDATA%\Timnot\gorevler.json

  • Application preferences (theme, language, notifications):
    %APPDATA%\Timnot\appsettings.json

These files are created solely on the user's device and are
never transmitted to any server, cloud service, or third party.

3. INTERNET ACCESS

Timnot does not require or use an internet connection.
No network requests are made.

4. SYSTEM PERMISSIONS

The App uses the following system functions ONLY upon
explicit user request:

  • Windows Task Scheduler (schtasks.exe):
    Only to create a "Wake Computer" scheduled task.

  • System tray icon:
    To run the application in the background.

  • Windows Startup Task (StartupTask):
    Only when the user enables "Start with Windows",
    visibly shown in Settings.

  • Power management (shutdown, restart, sleep, etc.):
    Only when a user-created task is triggered.

5. THIRD-PARTY SHARING

Timnot does not share any user data with third parties.
There is no integration with advertising networks, analytics
services, or data brokers.

6. CHILDREN'S PRIVACY

Since the App collects no personal data, it is safe for
users of all ages, including children under 13.

7. DATA SECURITY

All data is stored exclusively in the user's Windows account
AppData folder, accessible only by that user account.

8. POLICY CHANGES

When this policy is updated, the new version will be
published at the same address. Significant changes may be
announced on the Store listing page.
